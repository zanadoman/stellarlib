# stellarlib

**NOTE**: *stellarlib is a slop-free hand-crafted software*

## ⚙️ Prerequisites

```sh
cmake
clang
lld
sdl3
```

## 🚀 Build & Run

```sh
cmake -B build --toolchain cmake/x86_64-linux-gnu.cmake     # Configure
cmake --build build -t example                              # Build
build/example                                               # Run
```
